디지털 문구 처음 사용하시나요?

01

디지털 문구란?

디지털 문구는 종이로 배송되는 실물 상품이 아니라, 태블릿 필기 앱에서 사용할 수 있는 PDF 형식의 파일 상품입니다.

아이패드, 갤럭시 탭 등에서 굿노트, 노트쉘프, 삼성노트와 같은 앱으로 불러와 사용할 수 있습니다.

02

사용 전 준비물

  • 아이패드 또는 갤럭시 탭 등 태블릿 기기
  • 애플펜슬 또는 S펜
  • PDF를 불러올 수 있는 필기 앱
  • 다운로드한 디지털 문구 파일
03

디지털 서식 다운로드하기

상품 상세 페이지 또는 무료 콘텐츠 페이지에서 원하는 디지털 문구를 선택한 뒤 파일을 다운로드합니다.

다운로드 화면 예시 이미지가 들어갈 자리

04

굿노트에서 불러오기

  1. 다운로드한 PDF 파일을 파일 앱에서 확인합니다.
  2. 공유 버튼을 누릅니다.
  3. 앱 목록에서 굿노트를 선택합니다.
  4. 새로운 문서로 불러온 뒤 필기를 시작합니다.

굿노트 불러오기 화면 예시 이미지가 들어갈 자리

05

다른 필기 앱에서 사용하기

노트쉘프에서 사용하는 방법
  1. 노트쉘프 앱을 실행합니다.
  2. 가져오기 메뉴를 선택합니다.
  3. 다운로드한 PDF 파일을 선택합니다.
  4. 서식 위에 자유롭게 필기합니다.
삼성노트에서 사용하는 방법
  1. 삼성노트 앱을 실행합니다.
  2. PDF 가져오기 기능을 선택합니다.
  3. 저장된 PDF 파일을 불러옵니다.
  4. S펜으로 필기하며 사용합니다.
FAQ

자주 묻는 질문

실물 상품이 배송되나요?

아니요. 소하의 디지털 문구는 다운로드 후 사용하는 파일 상품입니다.

굿노트가 꼭 필요한가요?

아니요. PDF를 불러올 수 있는 노트쉘프, 삼성노트 등의 앱에서도 사용할 수 있습니다.

휴대폰에서도 사용할 수 있나요?

파일 확인은 가능하지만, 필기 사용은 태블릿 기기를 권장합니다.

다운로드한 파일을 다시 받을 수 있나요?

실제 서비스에서는 마이페이지 또는 주문 내역에서 재다운로드할 수 있도록 안내할 수 있습니다.
